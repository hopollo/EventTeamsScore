# EventTeamsScore

Key | Action
------------ | -------------
**NumpadSuppr** | **Start** the match
**Numpad4** | **+1pts** left team
**Numpad6** | **+1pts** right team
**Numpad1** & **Numpad7** | **-1pts** left team
**Numpad3** & **Numpad9** | **-1pts** right team
**NumpadEnter** | **End** the match
